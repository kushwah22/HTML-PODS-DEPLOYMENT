# 1. Create an HTML File
File: index.html
<!DOCTYPE html>
<html>
<head>
    <title>My HTML in Docker</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This HTML is running inside a Docker container!</p>
</body>
</html>
This HTML file is the content that will be served by the Nginx web server inside a Docker container.
